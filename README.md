# saidakhmad.dev

Personal site of **Saidakhmad Nuriddinov** — Autonomous Driving Simulation Engineer at Maum.ai (WoRV).

## Stack

- Next.js 16 (App Router) · React 19 · TypeScript
- Tailwind v4 (CSS-first config)
- shadcn/ui · base-nova style
- Spline (`@splinetool/react-spline`) — interactive 3D hero
- Framer Motion — scroll animations

## Develop

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Push to `main` on GitHub. Vercel auto-deploys. Preview deployments on every PR.
